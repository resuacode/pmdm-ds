# Navegación en Jetpack Compose

En Jetpack Compose, la navegación es un concepto fundamental que te permite moverte entre diferentes pantallas o destinos en tu aplicación. La navegación es importante para crear una experiencia de usuario coherente y fluida, y para permitir a los usuarios explorar y utilizar todas las funcionalidades de tu aplicación.    

En este documento, veremos cómo implementar la navegación en Jetpack Compose y cómo utilizar el componente `NavHost` para definir las rutas y destinos de tu aplicación. También veremos cómo pasar datos entre pantallas y cómo gestionar la navegación de forma segura y eficiente.   

## Configuración de la navegación

Para configurar la navegación en Jetpack Compose, necesitas agregar la dependencia de `navigation-compose` a tu archivo `build.gradle` y definir las rutas y destinos de tu aplicación en un archivo de configuración de navegación.
